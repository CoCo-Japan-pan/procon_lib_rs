Wavelet Matrix の索引として、ビットごとの累積和を追加することで、重み付きの点の静的な矩形区間和を高速に求めることができる。
数列の区間[l, r)のうちのx <= c < yを満たす数値の和を求めるrange_sumクエリは、各点の重みweight_listを (圧縮前の)y座標と同じものとすることで、矩形和のクエリに帰着できる。
