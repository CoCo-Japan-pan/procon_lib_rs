Mo’s algorithm
一般に二つのindex (x, y) をとるクエリ達に対して、xやyを+-1する変更をO(α)でできる場合
0 <= x <= nx, 0 <= y <= ny とすると全体でO(α\sqrt{nxnyQ})で処理できる (Q = クエリの数)
クエリ先読みが必要
index(x, y)が区間クエリ[x, y)に対応する場合がよくある使い方だが、 別にx <= yである必要はなく、一般に二つのindexをとるクエリ全般に使える
